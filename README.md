# Python Automação 🚀

Projeto desenvolvido durante a participação na Jornada Python da Hashtag Treinamentos, um evento intensivo com foco em aplicações práticas utilizando Python.

O objetivo deste repositório é reunir os projetos, estudos e aplicações desenvolvidas ao longo dos 4 dias do evento, abordando automação, análise de dados, machine learning e integração com inteligência artificial.

---

## 📚 Conteúdos abordados

### 🔹 Dia 1 — Automação com Python
- Automação de tarefas utilizando PyAutoGUI
- Controle de mouse e teclado
- Execução automática de processos repetitivos
- Ganho de produtividade com scripts Python

### 🔹 Dia 2 — Análise de Dados
- Manipulação de dados com Pandas
- Leitura e tratamento de arquivos
- Organização e análise de informações
- Geração de insights com dados

### 🔹 Dia 3 — Machine Learning
- Introdução à análise preditiva
- Treinamento de modelos
- Uso de algoritmos de Machine Learning
- Interpretação de resultados

### 🔹 Dia 4 — Inteligência Artificial + Streamlit
- Integração com modelos de IA
- Desenvolvimento de chatbot
- Criação de interface web com Streamlit
- Aplicações práticas de IA em Python

## ▶️ Como executar o projeto

### 1. Clone o repositório

```bash
git clone https://github.com/DarioLopes070/python-automacao.git
```

### 2. Acesse a pasta

```bash
cd python-automacao
```

### 3. Crie um ambiente virtual

```bash
python -m venv venv
```

### 4. Ative o ambiente virtual

#### Windows

```bash
venv\Scripts\activate
```

#### Linux/Mac

```bash
source venv/bin/activate
```

### 5. Instale as dependências

```bash
pip install -r requirements.txt
```

### 6. Execute o projeto desejado

Exemplo com Streamlit:

```bash
streamlit run app.py
```
